- Brief project description: This project allows users to discover Hiking Trails, Climbing Routes, and Camp Sites. The websites functionality not only allows users to sift, search, and filter for their desired search results, they also can add each item to a saved trip. The user can create as many trips as they'd like once logged in to an account. Since this project does not allow for writing to a json file, the website uses local web server data to store the data. Overall, users can seamlessly experience outdoor discovery through an immersive UI and responsive UX.
- Setup instructions: On any html file in this repo, open it with a live server. All pages on this website are accessible through any html file.
- Known limitations: The data provided in this website is not reflective of all existing trails, climbing routes, and camping sites that exist. Therefore, there may be data not displayed on this page that appears elsewhere.
